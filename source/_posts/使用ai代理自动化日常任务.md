<pre><code># 使用 npm 安装

# 或者使用其他包管理器
# bun install -g openclaw</code></pre>

<h3>配置</h3>
<pre><code># 初始化配置

# 启动网关

<h3>基本使用</h3>
<pre><code># 添加渠道

# 查看状态
