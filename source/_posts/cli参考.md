<h2>CLI概述</h2><p>OpenClaw CLI是管理OpenClaw实例的主要方式。</p><h2>常用命令</h2><ul><li>openclaw setup - 初始化配置</li><li>openclaw onboard - 启动新手引导</li><li>openclaw gateway - 管理Gateway网关</li><li>openclaw status - 查看状态</li><li>openclaw health - 健康检查</li><li>openclaw message send - 发送消息</li><li>openclaw channels login - 渠道登录</li><li>openclaw pairing - 配对管理</li><li>openclaw security audit - 安全审计</li></ul><h2>Gateway子命令</h2><ul><li>openclaw gateway start - 启动Gateway</li><li>openclaw gateway stop - 停止Gateway</li><li>openclaw gateway status - 查看状态</li><li>openclaw gateway install - 安装服务</li></ul><h2>配置命令</h2><ul><li>openclaw configure - 编辑配置</li><li>openclaw configure channels - 配置渠道</li><li>openclaw configure providers - 配置提供商</li></ul>
