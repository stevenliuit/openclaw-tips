<h2>什么是OpenClaw？</h2><p>OpenClaw是一个开源的AI助手，运行在你的自己的机器上。它在你的聊天应用中工作，可以学习你的习惯。2026年初，人们开始关注它，因为它在你的设备上运行，并且可以根据需要扩展。它还有一个龙虾吉祥物。</p><h2>是什么让OpenClaw与众不同？</h2><p>OpenClaw的工作方式更像一个同事，而不是另一个聊天窗口。三个特点让它脱颖而出：</p><ul><li><strong>完整的系统访问：</strong>与典型的聊天机器人不同，OpenClaw不仅仅局限于浏览器标签。它作为服务在你的电脑上运行。它可以读写文件、运行shell命令和执行脚本。它可以组织你的照片库、重构代码或设置cron任务，无需你的直接监督。</li><li><strong>持久记忆：</strong>在初始化时，它会询问你的名字、时区和一些偏好。OpenClaw会记住对话，注意你的行为模式并随时间建立上下文。用户将其记忆比作在Obsidian中保存每日笔记。助手将记忆存储为磁盘上的纯Markdown文件。</li><li><strong>主动心跳：</strong>OpenClaw可以自动唤醒来运行后台任务。它的"心跳"可以检查你的收件箱、发送早晨摘要或运行自动化任务，无需提示。这个功能让它更像一个后台服务而不是一个简单的程序。</li></ul><p>它可以控制浏览器、填写表单并从网站收集数据。它支持多种消息渠道，如Telegram、WhatsApp、Discord、Slack和iMessage。你可以在喜欢的应用中聊天。社区正在构建技能和插件。助手甚至可以生成新技能来完成任务。</p><h2>安装前准备</h2><p>OpenClaw对你的电脑有深度访问权限，所以最好在专用机器或虚拟专用服务器上设置它。你需要：</p><ul><li><strong>支持的操作系统：</strong>macOS、Linux，或通过WSL 2的Windows。树莓派或备用Mac mini可以托管它。</li><li><strong>Node.js ≥ 22：</strong>如果缺少Node，安装程序会安装它。</li><li><strong>API密钥：</strong>你需要为大型语言模型提供商（如Anthropic或OpenAI）提供凭证。如果需要网络搜索，获取Brave Search API密钥。</li></ul><h2>快速安装</h2><p>最简单的入门方式是使用官方安装脚本。它安装所有依赖项、设置CLI并运行初始化向导：</p><pre><code># 在macOS或Linux上
curl -fsSL https://openclaw.ai/install.sh | bash

# 在Windows上(PowerShell)
iwr -useb https://openclaw.ai/install.ps1 | iex</code></pre><p>脚本如果缺少Node会安装它。它下载OpenClaw并运行初始化向导。完成后，用以下命令检查CLI：</p><pre><code>openclaw doctor</code></pre><p>用以下命令查看网关状态：</p><pre><code>openclaw status
# 或
openclaw health</code></pre><h3>替代安装方法</h3><p>如果你更喜欢自己动手或想修改代码，还有其他选项：</p><ul><li><strong>全局npm安装：</strong>如果已经安装了Node，运行<pre><code>npm install -g openclaw@latest</code></pre>如果<pre><code>sharp</code></pre>依赖因全局<pre><code>libvips</code></pre>失败，使用<pre><code>SHARP_IGNORE_GLOBAL_LIBVIPS=1 npm install -g openclaw@latest</code></pre>。安装后，运行<pre><code>openclaw onboard --install-daemon</code></pre></li><li><strong>从源码安装：</strong>克隆仓库，安装依赖，构建UI并运行初始化向导：<pre><code>git clone https://github.com/openclaw/openclaw.git
cd openclaw
pnpm install
pnpm ui:build
pnpm build
pnpm run openclaw onboard</code></pre>这种方法适合想了解OpenClaw内部的开发者。</li></ul><h2>初始化你的龙虾</h2><p>安装OpenClaw后，运行初始化向导。输入：</p><pre><code>openclaw onboard</code></pre><p>运行向导来配置助手。它会引导你创建本地网关和设置工作区。它连接消息渠道并安装系统服务。有两条路径：</p><ul><li><strong>快速开始：</strong>选择合理的默认值。它使用绑定到<pre><code>localhost</code></pre>的本地网关、默认工作区文件夹、端口18789和自动生成的认证令牌。它不使用Tailscale，并为新联系人使用白名单。</li><li><strong>高级：</strong>让你自定义更多内容。你可以设置工作区位置、网络绑定、守护进程安装以及选择启用哪些渠道和技能。</li></ul><h3>向导配置什么？</h3><p>在初始化过程中，你会被提示：</p><ul><li><strong>模型提供商和认证：</strong>选择Anthropic API密钥、OpenAI Codex订阅、纯OpenAI API密钥或其他提供商如Moonshot或MiniMax。</li><li><strong>工作区位置：</strong>默认情况下，它将文件存储在<pre><code>~/.openclaw/workspace</code></pre>下。这个文件夹包含配置文件、记忆笔记和技能。</li><li><strong>网关设置：</strong>选择端口。决定网关是绑定到<pre><code>localhost</code></pre>还是特定IP。决定是否通过Tailscale暴露它。即使在环回上，也要保留认证令牌，这样客户端必须认证。</li><li><strong>消息渠道：</strong>在初始化期间连接你的聊天服务。配对后，像往常一样发送消息。OpenClaw支持WhatsApp、Telegram、Discord、Slack、Signal、iMessage等。开启文本转语音后，它可以发送音频消息。</li><li><strong>守护进程安装：</strong>在macOS上，它安装一个<pre><code>LaunchAgent</code></pre>；在Linux或WSL 2上，它安装一个<pre><code>systemd</code></pre>用户单元，这样助手在你退出登录后仍能继续运行。</li><li><strong>技能：</strong>向导可以安装一些技能，无需额外步骤。你以后可以通过<pre><code>openclaw skills install &lt;package&gt;</code></pre>添加更多。</li></ul><p>如果跳过初始化（例如传递<pre><code>--no-onboard</code></pre>标志），你可以随时运行<pre><code>openclaw onboard --install-daemon</code></pre>。</p><h3>配置网页工具和搜索</h3><p>OpenClaw有内置的网页浏览和搜索工具。要启用Brave Search，运行：</p><pre><code>openclaw configure --section web</code></pre><p>然后粘贴你的Brave Search API密钥。没有密钥时，助手会回退到基本的网页获取工具。</p><h2>与助手对话</h2><p>网关运行后，你可以通过几种方式与你的龙虾聊天：</p><ul><li><strong>控制UI：</strong>要通过网页仪表板聊天，初始化后运行<pre><code>openclaw dashboard</code></pre>。它会启动一个本地网页界面。</li><li><strong>消息应用：</strong>在初始化期间连接你的聊天服务。配对后，像往常一样发送消息。OpenClaw支持WhatsApp、Telegram、Discord、Slack、Signal、iMessage等。开启文本转语音后，它可以发送音频消息。</li><li><strong>命令行：</strong>使用<pre><code>openclaw message &lt;channel&gt; &lt;recipient&gt; &lt;message&gt;</code></pre>发送命令，或使用<pre><code>openclaw browser &lt;url&gt;</code></pre>让助手浏览网站。还有查看记忆、安装技能和检查健康状况的命令。详见CLI参考。</li></ul><h2>OpenClaw能做什么？</h2><p>因为OpenClaw有系统访问和持久记忆，它可以处理很多任务。这里有一些入门建议：</p><ul><li><strong>收件箱和日历管理：</strong>让它监控你的电子邮件、筛选消息并起草回复。它可以通过检查你的日历并发送确认消息来重新安排会议。</li><li><strong>每日简报和报告：</strong>设置早晨摘要并通过你的聊天应用发送，包含文本和音频。</li><li><strong>文件整理和备份：</strong>让它比较本地文件夹和云备份、移动文件或清理下载目录。</li><li><strong>自定义自动化：</strong>使用技能系统构建自定义工作流。例如，你可以创建一个使用Whipser API转录语音消息的技能，或设置一个监视RSS订阅源并触发工作流的cron任务。</li><li><li><strong>代码和脚本任务：</strong>它可以编写和执行脚本、重构代码或设置开发环境。</li></ul><p>当你需要新东西时，让助手"安装一个X技能"或自己写一个。OpenClaw可以搜索技能中心并安装或创建代码。</p><h2>注意安全和成本</h2><p>让AI代理访问你的机器是有风险的。配置不当的网关或未验证的技能可能会暴露敏感文件或API密钥。在你控制的机器上运行向导，保持软件更新，并与已知的联系人配对。从只读权限开始，随着获得信任而增加权限。</p><p>OpenClaw可以免费下载，但它使用付费API进行模型推理、网络搜索和语音功能。如果你让助手一直运行或使用大型模型，这些成本可能会增加。在API提供商的仪表板中监控使用情况并设置限制。</p><blockquote><p>OpenClaw不仅仅是一个聊天机器人。它在你的机器上运行，使用大型语言模型。它会记住你的互动，可以安排任务。安装过程很短，初始化向导会引导你完成配置。有了关怀和深思熟虑的使用，OpenClaw可以成为一个可靠的私人助手。</p><p>无论你是开发者、效率爱好者还是只是好奇，OpenClaw都可能让你感兴趣。命名你的新助手，避免在与敏感文档相同的机器上运行。</p></blockquote>
