<h2>什么是 OpenClaw？</h2><p><img src="/images/openclaw-152-1.png" alt="什么是 OpenClaw" /></p><p><a href="https://openclaw.ai/">OpenClaw</a> 是一个开源的 AI 智能体框架。与普通的聊天机器人不同，OpenClaw 可以在你的电脑上执行实际的操作。</p><p>例如，它可以：</p><ul><li>运行终端命令</li><li>读取和编辑文件</li><li>自动化工作流程</li><li>控制浏览器</li><li>使用本地记忆记住任务</li></ul><p>OpenClaw 充当 LLM 推理模型与操作系统之间的桥梁。</p><h2>为什么使用 Ollama 运行 OpenClaw？</h2><p>使用 Ollama 运行 OpenClaw 可以让你拥有完全本地的 AI 智能体。</p><ol><li><strong>完全隐私</strong>：所有数据都保存在你的电脑上</li><li><strong>无需 API 费用</strong>：你不需要 OpenAI 或云服务提供商</li><li><strong>更快的性能</strong>：本地模型消除了网络延迟</li><li><strong>持久记忆</strong>：OpenClaw 将对话存储在本地 Markdown 文件中，实现长期记忆</li><li><strong>消息接口</strong>：你可以通过以下方式控制 OpenClaw：</li></ol><ul><li>Telegram</li><li>Slack</li><li>WhatsApp</li></ul><p>这允许你从手机触发工作流程。</p><h2>OpenClaw 的最佳本地模型</h2><p><img src="/images/openclaw-152-2.png" alt="OpenClaw 的最佳本地模型" /></p><p>选择合适的本地模型对于可靠的智能体行为非常重要。</p><p>为了可靠的工具使用，请使用 14B 或更大的模型。小模型在执行多步骤命令时经常会失败。</p><h2>如何安装 OpenClaw 和 Ollama</h2><h3>第一步 — 安装 Ollama</h3><p>安装 Ollama：</p><pre><code>curl -fsSL https://ollama.com/install.sh | sh</code></pre><p>验证安装：</p><pre><code>curl http://localhost:11434/api/tags</code></pre><p>然后从 <a href="https://ollama.com/">ollama.com</a> 网站下载以下模型之一：</p><ul><li><a href="https://ollama.com/library/qwen3-coder">qwen3-coder</a> — 针对编码任务优化</li><li><a href="https://ollama.com/library/glm-4.7">glm-4.7</a> — 强大的通用模型</li><li><a href="https://ollama.com/library/gpt-oss:20b">gpt-oss:20b</a> — 平衡的性能和速度</li><li><a href="https://ollama.com/library/gpt-oss:120b">gpt-oss:120b</a> — 改进的能力</li></ul><p>命令示例：</p><pre><code>ollama run qwen3-coder</code></pre><h3>第二步 — 安装 OpenClaw</h3><p>安装 OpenClaw：</p><pre><code>curl -fsSL https://openclaw.ai/install.sh | bash</code></pre><p>使用以下命令通过 Ollama 启动 OpenClaw：</p><pre><code>ollama launch openclaw</code></pre><p><img src="/images/openclaw-152-3.png" alt="使用 Ollama 运行 OpenClaw" /></p><h3>视频教程</h3><p><strong>在 YouTube 上观看：</strong> <a href="https://youtu.be/dRXWkHSTJG4">如何设置 OpenClaw 和 Ollama</a></p><h2>安全警告：「内核模块」警告</h2><p>截至 2026 年 3 月的安全更新，OpenClaw 的广泛权限是一把双刃剑。因为它在内核/操作系统级别运行：</p><p><strong>禁用网页搜索：</strong> 对于完全本地的工作流程，将搜索切换为 <code>false</code>。在配置中确保没有数据片段被发送到搜索引擎。</p><p><strong>审计你的日志：</strong> OpenClaw 将每个操作保存在本地日志中。定期检查这些日志，确保你的智能体没有执行「幽灵操作」。</p><p><strong>人工介入：</strong> 对于敏感命令（如 <code>rm -rf</code> 或发送外部邮件），始终将工具权限设置为「询问」。</p><h2>结论</h2><p>如果你按照本指南中的步骤操作，你现在应该已经有一个使用本地模型运行的 OpenClaw 设置。</p><p>试试看，尝试不同的模型，看看你可以自动化什么样的工作流程。</p><p>如果你发现有趣的内容，请随时分享。我总是很好奇人们如何使用这些工具。</p><p>干杯，proflead！;)</p>
