<h2>Cron任务概述</h2><p>OpenClaw支持类似cron的定时任务，允许智能体在特定时间自动执行操作。</p><h2>配置</h2><pre><code>{ "cron": { "tasks": [ { "id": "daily-briefing", "schedule": "0 8 * * *", "action": { "type": "agent", "agentId": "main", "message": "给我今天的简报" } } ] } }</code></pre><h2>时间格式</h2><p>使用标准cron格式：</p><ul><li>分 时 日 月 星期</li><li>0 8 * * * — 每天早上8点</li><li>0 9 * * 1-5 — 工作日早上9点</li></ul><h2>任务类型</h2><ul><li>agent — 运行智能体轮次</li><li>send — 发送消息</li><li>webhook — 调用webhook</li></ul><h2>心跳vs Cron</h2><p>了解何时使用哪种机制。</p><h2>示例</h2><ul><li>每日天气摘要</li><li>定时健康检查</li><li>定期数据备份</li></ul>
