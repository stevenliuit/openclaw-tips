<h2>快速开始</h2><p>OpenClaw需要Node 22或更新版本</p><h2>安装</h2><p>npm install -g openclaw@latest</p><h2>新手引导并运行Gateway网关</h2><ol><li>openclaw onboard --install-daemon</li><li>openclaw channels login</li><li>openclaw gateway --port 18789</li></ol><h2>从源码安装（开发）</h2><p>git clone https://github.com/openclaw/openclaw.git && cd openclaw && pnpm install && pnpm ui:build && pnpm build && openclaw onboard --install-daemon</p><h2>发送测试消息</h2><p>openclaw message send --target +15555550123 --message "Hello from OpenClaw"</p>
